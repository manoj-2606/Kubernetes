# Day 07 — Goals

1. Understand DaemonSets — what they are, when to use them, and how they differ from Deployments
2. Deploy a DaemonSet and observe one Pod per node behavior
3. Understand Jobs — run-to-completion workloads with guaranteed execution
4. Deploy a Job and observe completion behavior
5. Understand CronJobs — scheduled Jobs with concurrency and history controls
6. Deploy a CronJob and observe scheduled execution
7. Understand NetworkPolicy — default allow behavior and how to enforce deny
8. Implement a default-deny NetworkPolicy and explicit allow rules
9. Verify NetworkPolicy enforcement with curl tests between namespaces
