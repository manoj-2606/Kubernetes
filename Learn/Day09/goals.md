# Day 09 — Goals

1. Create a new Azure DevOps project and GitHub repo dedicated to AKS
2. Set up a new service connection with OIDC authentication for the AKS project
3. Bootstrap remote state storage inside the pipeline
4. Provision an AKS cluster using Terraform with production-grade configuration
5. Configure a dedicated system node pool and user node pool
6. Enable Workload Identity and OIDC issuer on the AKS cluster
7. Connect kubectl to the provisioned AKS cluster
8. Verify the cluster is healthy and ready for Day 10 workloads
