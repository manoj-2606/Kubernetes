# Day 11 — Goals

1. Create an Azure Container Registry (ACR) and attach it to the AKS cluster
2. Write a simple custom Helm chart for a sample application
3. Build a Docker image and push it to ACR via Azure DevOps pipeline
4. Deploy the application to AKS using Helm from the pipeline
5. Implement the full CI/CD loop — code push triggers build, push, and deploy
6. Understand GitOps — how it differs from pipeline-driven deployments
7. Understand ArgoCD and Flux conceptually and when to use each
8. Complete the Kubernetes learning track with a portfolio-ready deployment
