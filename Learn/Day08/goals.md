# Day 08 — Goals

1. Understand Ingress — how it differs from LoadBalancer and NodePort Services
2. Deploy an Ingress controller and configure host-based and path-based routing
3. Understand HPA — how it scales Pods based on CPU and memory metrics
4. Deploy an HPA and observe automatic scale-up and scale-down behavior
5. Understand Taints and Tolerations — how to repel or attract Pods to nodes
6. Apply a taint to a node and observe Pod scheduling behavior
7. Understand Node Affinity and Pod Affinity — preference vs requirement
8. Configure nodeAffinity rules and observe scheduler behavior
9. Understand PodDisruptionBudget — protecting availability during voluntary disruptions
