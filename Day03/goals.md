# Day 03 — Goals

1. Understand why configuration must be separated from container images
2. Create a ConfigMap and consume it as environment variables inside a Pod
3. Create a ConfigMap and consume it as a mounted volume inside a Pod
4. Create a Secret and consume it safely inside a Pod
5. Understand the difference between env injection and volume mounting
6. Understand what happens to a running Pod when a ConfigMap is updated
7. Understand why Secrets are not truly secret by default and what to do about it
